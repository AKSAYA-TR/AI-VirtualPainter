🖐️ AI Virtual Painter
An interactive drawing application that uses Hand Gesture Recognition to paint in the air.

🚀 Features
Hand Tracking: Real-time 21-point landmark detection using MediaPipe.

Dual Modes: * Selection Mode: Raise two fingers to choose colors or the eraser from the header.

Drawing Mode: Raise only the index finger to draw smoothly on the screen.

Custom Canvas: Uses NumPy bitwise operations to merge the drawing canvas with the live camera feed.

Dynamic Header: Interactive UI that updates based on the selected tool.

🛠️ Tech Stack
Python 3.12

OpenCV (Image processing and UI)

MediaPipe (ML-based hand tracking)

NumPy (Canvas matrix manipulation)

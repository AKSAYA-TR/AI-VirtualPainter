# AI-VirtualPainter
AI Virtual Painter | Real-time air-painting app using Python 3.12, OpenCV &amp; MediaPipe. Features 21-point hand landmark tracking for seamless Drawing &amp; Selection modes. Uses NumPy bitwise operations to merge a digital canvas with live video. Select colors or erase with intuitive hand gestures. 🖐️🎨 #ComputerVision #Python #AI
